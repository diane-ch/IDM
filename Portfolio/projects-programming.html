EEG / Brain-Computer Interface

An electroencephalogram (EEG) is a recording of brain activity. In collaboration with an engineer in Nantes University Hospital, I completed a project for EEG-based topological functional connectivity analysis for motor mental imagery.

  • Tasks:
    • proposing a method for visualizing graph topology
    • study of the dynamics of connectivity over time during and IMM task
    • Validate the methods on a public database
  • Key topics:
    • Data processing
    • Machine learning
    • Graph topology
  • Programming language: Python
